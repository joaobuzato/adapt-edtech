# adapt-edtech

Tecnical Test to Adapt Edtech

## How to run the project

1. Clone the repository

### Server

2. Run `cd server`
3. Run `npm install`
4. Run `npm start`

### Client

5. Run `cd client`
6. Run `npm install`
7. Run `npm start`

Execute one on each terminal and access `http://localhost:3000/` on your browser.
